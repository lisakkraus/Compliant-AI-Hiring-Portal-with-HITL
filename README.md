# Compliant AI-Hiring Portal with HITL

This is a workflow for an AI hiring portal with governance that achieves regulatory compliance through HITL

* New regulations consider hiring a high-risk action.
* High-risk actions/data regulations require two human reviewers.
* New regulations also require actively working to prevent/detect automation bias.
* This workflow shows how AI can be used to sort applications to speed up workflows while remaining compliant with new regulations.
* Currently, most AI hiring systems automatically reject outliers, which could be a surprisingly good match. This system ensures they are batched separately for human review.
* The system also shows a sample dashboard.
